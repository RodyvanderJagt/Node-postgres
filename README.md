# Node-postgres
## RESTful API with Node.js, Express, and PostgreSQL

This project creates an CRUD RESTful API in a Node.js environment running on an Express.js server which is hooked up to an active PostgreSQL database.

## Seeding Database with thousands of users using Knex.js and Faker.js
The database is filled with thousands of random users generated by faker.js. Knex.js builds the SQL queries used to create and fill the database.
